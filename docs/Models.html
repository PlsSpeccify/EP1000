Creating models with FUSION 360

Following up with previous topic on CAD, FUSION 360 can be used to turn sketches into models. There are 3 ways to do so:

  • Extrusion
    -creating a 2D outline in one plane and then extruding or extending the outline of the object in the axis perpendicular to that plane.
  • Revolution
    -creating a 2D outline in one plane/shape and then revolving the outline/shape a number of degrees in the perpendicular plane. This method is ideal for creating cylindrical objects.
  • Sculpting
    -This method is akin to playing with plasticine. We start with a defined object e.g. block, cylinder, sphere and then by pushing and pulling surfaces we are able to create a 3D model of the object.

In this page I document several models I made with FUSION 360.

  • Nametag
  • Fidget Spinner
  • Knight Chess piece
  • Laptop stand

Nametag

Specifications:

  • The nametag should not exceed 30mm X 70mm.
  • The key ring hole must have a diameter of at least 2.5 mm with neck width of at least 3.5 mm for strength
  • Do not make the keytag thicker than 4 mm

I first start by drawing the nametag with the sketch function.

I created the sketch on the X-Y plane as I want it to be lying flat.

After that I start the extrusion. I first extrude the center piece by 2mm.

I next extrude the outer shell and the ring by 4mm.
(You can also extrude the whole body then use shell instead of extruding 2 separate areas: all you need to do is select the area and the thickness of the sides.)

Lastly I extrude the text.
Extrude function allows you to either join objects or cut the object. Here I extruded the text by 3mm joined.

I then use fillet to round the edges of the ring hole and the body to remove sharp edges.

Fidget Spinner

Specifications:

  • An ABEC 608 Bearing of dimensions 8 x 22 x 7 mm for the rotation device and M8 Stainless Steel hex nuts as counter-weights will be used.
  • The radius of your fidget spinner should not exceed 50mm.
  • Maximum thickness of your fidget spinner 8 mm to accommodate the bearing and the M8 nuts.

My spinner design will be based on a Biohazard symbol.

First I used construction line to mark out the max diameter of the spinner (100mm).

In the center I draw a circle with the diameter 22mm of the bearing and create another circle with an offset or 2mm.

I draw one part of the main design.

Using circular pattern I copy the design part and paste it around the origin point. This saves time from drawing 3 of the same part.

Now extrusion: I extrude the center ring by 8mm in symmetric so the main desing will be at the center.

I then extrude the main design by 4mm in symmetric.

Next is to add the hex weights: I draw a construction line to one corner and draw a hexagon with circumscribe distance with 6.5mm. Then an outer hexagon offest to 2mm.

Using circular pattern to repeat 2 more hexagons on each corners.

I first join extrude the outer area on two sides: 2mm on top and 6mm below.

I then cut extrude the inner hexagon to create a hole for the nuts to fit.

Knight Chess piece

For the chess piece I will be drawing the sketch on the YZ plane.

I create a box as the border for the chess piece.

I inserted a knight image to trace over.

For the base I drew a vertical line at the center of the image and drew half of the base.

Hiding the canvas you can see the final sketch.

Starting with the base, I used Revolve. Select the base sketch and then select the point of rotation.

I then use shell to hollow the base to minimise material usage, setting the thickness to 4mm.

Then I extrude the knight by 4mm, and the back part by 3mm, extruding symmetrically.

Laptop Stand

Starting with the leg, I first draw a rough outline of it on the YZ plane.

I constraint the slanted angles to be parallel, then set dimensions to fully constraint the sketch.

Next I set parameters for thickness and leg-width which can be found under the Modify tab. This makes it easier to adjust material thickness and width without having to redefine every body part.

To use the parameters, type the parameter name in the object dimension. As you change the parameter value the body dimensions will update and change.

I fillet the edges of the leg, then copied the body a legwidth distance away.

Next I created a new component "Top". I project the slanted side of the legs to make it more visible. I then create a sketch on the side and draw a rectangle.

I then add dimensions to constraint the sketch.

I want to add tabs so I can fit the parts together, so I draw a small rectangle at one corner, above the leg piece and added dimensions.


To add tabs to the other corners, I draw a center line on horizontally and vertically. Then I use the mirror tool to project the rectangle to each corners.

I extrude the top by negative "thickness", so the part will cut into the legs, creating it as a new body.

To cut the tabs, I use the combine tool. First selecting the legs as the Target body and the top as the Tool body. I use the cut operation while selecting keep tools.

The end result is tabs appearing on the legs.

Next I want to create extra supports for the legs. First I offest a plane so the supports can be drawn in between the legs.



I sketch a rectangle support on the offest plane and constraint it.

As I want to be able to slot the support in, when extruding I leave a gap at where the legs are.

I then create a midplane to mirror the support to the other side. Similar to before, I use the combine tool to create the slot, with the legs as the Target body and the support as the Tool body, using cut while keeping the tool.

To make it easier to slot in the tabs, I fillet the tabs and slot edges.

To add texture, I used Appearance under Modify tab. Simple find for a texture of your choice and drag it onto the part. Here I give the stand an oak wood texture.

Laser cut box

I first start by making parameters. This will make it easier to modify and adjust dimensions without the need of redrawing. Under the modify tab, click parameters. You can define the parameter name, unit and expression. You can also define parameters using equations and existing parameter variables.

I will start with the base. Create a new component o organise the different parts. To use the parameters, simply type in the parameter name when defining the dimensions.

To make the base offset, I draw a center rectangle and define the offsets.

I then extrude with parameter "thick"

I next create a new sketch and draw a small rectangle touching the inner rectangle for the slot tabs.

I then extrude cut and used rectangular pattern to repeat the cut along the length.

I create a new component for the back and create an offset plane such that it touches the surface of the tabs.

I draw a rectangle, dimension it with "height" and "length" and extrude it by "thick"

I create a new sketch at the corner and draw a small rectangle of "Htabwidth" tall and "Htabwidth" above the bottom base.

Similar to the base tabs I cut extrude and use rectangular pattern to copy the cut along the height.

To create the bottom tabs, I used the combine tool, selecting the back as the Target body and the base as the tool body. Selecting cut operation and keep tool.

Next I create a new component and extrude the back with offset such that it extrudes at the other side.

To cut the slots in the base I use combine tool, this time the front as the Tool body.


Now for the front, I create a new component and create new sketch on the surface of the back piece tabs. I then draw a rectangle and collinear constraint to the corners of the base, front and back.


Then I extrude inwards as a new body.


I forgot to cut the width tabs so I went back to the base component and repeat the tab cutting step: draw rectangle, dimension it, extrude cut and rectangular pattern.




Then to create tabs on the right piece I use combine tool cut, with the Right piece as the target body.


I create new component and extrude offset to the left side. Lastly I use combine cut to create the slot holes in the base.



Pulling the pieces apart you can inspect the slots have been cut properly.


I then give it a wood texture for aesthetics.
